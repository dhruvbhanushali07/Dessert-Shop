## ‚ú® Key Features & Technical Highlights

This application was developed with a focus on interactive UI design and efficient state management:

| Category | Highlight | Description |
| :--- | :--- | :--- |
| **User Interface** | **Dynamic & Responsive UI** | Built with **React** components for a modular, maintainable, and dynamic interface. |
| **Design & Styling** | **Mobile-First Design** | Styled using **Tailwind CSS** for rapid development, utility-first styling, and optimal usability across all screen sizes. |
| **State Management** | **Centralized State** | Utilizes the lightweight **React Context API** for global state management, handling cart contents and real-time updates. |

| **E-commerce Core** | **Interactive Shopping Cart** | Features a fully functional cart allowing users to add and remove products, and dynamically calculate totals and quantities. |

## üõ†Ô∏è Installation and Setup

To run this project locally, follow the steps below. You will need **Node.js** and **npm** installed.

### 1\. Clone the Repository

```bash
# Clone the repository
git clone [YOUR_REPO_URL_HERE]

# Navigate into the project directory
cd dessert-shop
```

### 2\. Install Dependencies

```bash
# Install all required npm packages
npm install
```

### 3\. Run the Development Server

```bash
# Start the development server
npm run dev
```

The application will launch in your browser, typically at `http://localhost:5173/`.
